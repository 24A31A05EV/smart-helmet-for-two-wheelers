# Smart Helmet for Two-Wheelers

## Problem
Many riders do not wear helmets, leading to accidents and unsafe riding.

## Solution
A smart helmet system that ensures safety by:
- Detecting helmet usage
- Preventing bike start without helmet
- Sending emergency alerts

## Features
- Helmet detection (simulated)
- Bike ignition control
- Emergency alert system

## How to Use
1. Click "Wear Helmet"
2. Click "Start Bike"
3. Try emergency alert button
